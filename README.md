# National Quemoy University (NQU) Website

This is a modern, premium redesign of the National Quemoy University website, built with **React**, **Tailwind CSS v4**, and **Framer Motion**.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or pnpm

### Installation
```bash
npm install --legacy-peer-deps
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## 🌍 Deployment on Vercel

1. **Push to GitHub**: Initialize a git repository and push your code to GitHub.
2. **Connect to Vercel**:
   - Go to [vercel.com](https://vercel.com)
   - Click "Add New Project"
   - Import your NQU repository
   - Vercel will automatically detect the Vite settings.
   - Click **Deploy**.

## ✨ Features
- **Responsive Design**: Mobile-first approach for all academic sections.
- **Glassmorphism UI**: High-end aesthetic with transparent blurred elements.
- **Animations**: Smooth scroll reveals and micro-interactions using Framer Motion.
- **Dark Mode**: Integrated support for system-based or manual dark/light toggling.
- **Multi-language Ready**: Built-in logic for English and Traditional Chinese support.

## 🛠️ Tech Stack
- **Frontend**: Vite + React + TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router
